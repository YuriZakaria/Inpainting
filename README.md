# Image Inpainting
Image reconstruction by Inpainting of uniformly distorted images using a simple convolution neural network CNN utilizing PyTorch 1.1 as a submission for the Python Programming 2 project at the Johannes Kepler University Linz.

## Dataset
Collaboratively Gathered images from over 300 different students with an average of 50 photos per student.

## Usage
Update the config.json file to either "train" or "test" mode and the input/output dirs as well for the desired mode.
Dont forget to include your dataset folders
